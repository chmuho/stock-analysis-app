# 개인 투자 분석 도우미

초보자가 아무 종목이나 충동적으로 사지 않도록 돕는 로컬 웹앱입니다. 종목 분석, 세후 손익, 배당 예상, 모의투자 기록을 한 화면에서 봅니다.

## 실행

PowerShell에서 이 폴더로 이동한 뒤 실행합니다.

```powershell
python app.py
```

브라우저에서 아래 주소를 엽니다.

```text
http://127.0.0.1:8501
```

## 현재 기능

- 미국/한국 종목 가격 데이터 조회
- 미국 대형주 60개, ETF 20개, 배당 후보 20개 빠른 스캔
- 한국 대표주는 `삼성전자`, `NAVER`, `SK하이닉스`처럼 종목명 입력 가능
- 종목 검색창에서 이름/티커 검색 후 바로 분석
- 종목별 지금 진입 판단: 진입 가능/분할만/대기/피하기
- 진입 판단별 버튼 필터
- 미국 후보 120개 확장 스캔
- 미국 상장 종목 목록 기반 전체 샘플 300개 스캔
- 표시/전체 카운터로 필터 적용 후 개수와 실제 분석 개수 구분
- 현재 거래량과 20일 평균 대비 거래량 표시
- 최근 뉴스 헤드라인 기반 호재/악재/중립 빠른 요약
- 한국/미국 선택에 따라 세금 안내와 안전장치 화면 자동 전환
- 1개월/1년/5년 수익률, 이동평균선, RSI, 거래량 분석
- 위험/주의/관찰/상대적 양호 판정
- ETF 중심, 배당 후보, 성장주 후보 빠른 목록
- 최근 1년 배당금 기반 배당수익률과 세후 배당 추정
- 최근 1개월 수익률 비교 그래프
- 위험점수와 배당수익률 산점도
- 동일비중 투자 시 세후 연 배당 예상 그래프
- 상대적으로 먼저 볼 후보 TOP 5
- 투자금 입력 기반 분산 배분 예시
- 미국주식 세후 손익 계산
- 브라우저 저장소 기반 모의투자 기록
- 수익률, RSI, 배당률, 위험점수 필터

## 세금 가정

앱의 세금 계산은 초보자용 단순 모델입니다.

- 해외주식 매도차익: 연간 손익통산 후 250만원 기본공제, 초과분 22%
- 미국 배당: 15% 원천징수 후 입금된다고 가정
- 국내 배당: 15.4% 원천징수로 단순 가정
- 금융소득이 커지면 금융소득종합과세, 외국납부세액공제, 환율 적용 방식 등을 별도로 확인해야 합니다.

## 종목 코드 예시

- 미국 ETF: `VOO`, `VTI`, `SPLG`, `SCHD`, `QQQM`
- 미국 대형주 스캔: 빠른 목록에서 `미국 대형주 60` 선택
- 미국 배당주: `KO`, `PEP`, `JNJ`, `PG`, `MCD`
- 미국 성장주: `AAPL`, `MSFT`, `NVDA`, `TSLA`
- 한국 주식: `005930.KS`, `000660.KS`, `035420.KS`
- 한국 종목명 입력: `삼성전자`, `SK하이닉스`, `NAVER`, `카카오`

데이터는 Yahoo Finance 공개 엔드포인트를 사용합니다. 지연, 누락, 오류가 있을 수 있으므로 실제 투자 전에는 증권사 앱, 공시 자료, 세무 안내를 다시 확인해야 합니다. 이 앱은 매수/매도 추천이나 수익 보장을 하지 않습니다.
